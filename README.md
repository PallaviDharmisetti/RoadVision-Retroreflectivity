# 🚀 RoadVision System

## Intelligent Highway Retroreflectivity Monitoring

This project is an AI-based system that analyzes road images, videos, or live feeds to detect road features and evaluate their visibility using a Reflection Index.

---

## 🔍 Features

- Real-time road analysis
- Lane, signboard, and road stud detection
- Reflection Index (0–100)
- Weather detection (Rain, Fog, Mist, Night)
- Remaining life prediction
- Repair urgency estimation
- Works on image, video, and webcam

---

## 🧠 Technologies Used

- Python
- OpenCV
- NumPy
- Computer Vision
